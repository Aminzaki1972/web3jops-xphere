
# Web3Jops - Decentralized Hiring on Xphere

> Connecting MENA developers to global Web3 jobs with instant crypto payments & on-chain reputation

## 🚀 Problem
80% of MENA freelancers lose 20-30% of income to fees, delays, and disputes on Web2 platforms like Upwork. No true crypto-native hiring solution exists for the region.

## 💡 Solution  
Web3Jops is a decentralized job marketplace built on Xphere featuring:
- Smart Contract Escrow: Funds locked on-chain, released automatically on milestone approval
- Skill NFTs: Verifiable on-chain resumes that developers own
- Native DID Integration: Trustless identity using Xphere's native DID system
- 1-Second Payments: Leverage Xphere's 4,000+ TPS for instant crypto payouts

## ⛓️ Why Xphere
- Speed: 4,000+ TPS with 1-second finality vs 15s on Ethereum
- Cost: <$0.001 gas fees enables micro-task payments
- Identity: Native DID support for reputation without KYC
- EVM Compatible: Easy deployment using existing Solidity tools

## 🛠️ Tech Stack
- Smart Contracts: Solidity, Hardhat, Xphere EVM
- Frontend: Next.js 14, Wagmi, RainbowKit, TailwindCSS
- Storage: IPFS for job metadata + Skill NFT images
- Identity: Xphere DID SDK
